# bmath-pyqs
## This is a backup of bmath Previous year questions as website is not reliable. Everything is taken from [here](https://www.isibang.ac.in/~statmath/oldqp/B1Y.html)
Every file will be of the form **[(Subject)(Number)(X)(20XX-20YY)]**
If X is
1. empty: Endsem paper
2. BP: Back paper (endsem)
3. MS: Midsem paper
4. Some paper may end with 's' after the year, those are for supplementary midsem papers <br>
I'll add the B3 old curriculum papers afterwards. If I have made some error/missed PYQs, do contact me at adityateraiyaself@gmail.com <br>

A similar repo for [MMath here.](https://github.com/adi42wastaken/mmath-pyqs)
